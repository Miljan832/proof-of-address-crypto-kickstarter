Project Oblio Kickstarter

Our cryptocurrency kickstarter will require each user to donate in amounts of 0.1 ETH and will only accept their donations if they include a valid name and address.

Kickstarter

Wallet Address

The Kickstarter launches on November 5th, 2018 at 12:00 UTC. Any donations sent to the address before this time (or block number) will be returned to the user.

The user receives 20 OBL (name of token) for every 0.1 ETH that they donate. If they donate 0.1999 ETH, they only receive 20 OBL.

Each user has a separate "karma" score for every 0.1 ETH that they donate. So a user who donates 0.1 ETH has a karma score of 1. A user who donates 0.2 ETH has a karma score of 2.

The user must include the following in their transaction's data field: Name, Street Number, Street Address, Zip Code, and "terms". If "name" is not a string, the contract returns an error. If street number is not a number, the contract returns an error. If zip code is not at least 5 digits, the contract returns an error.. If any of these fields are missing, the contract returns the transaction and emits an error. If "terms" is not a "1" (indicating they accepted the terms and conditions), the contract emits an error.
